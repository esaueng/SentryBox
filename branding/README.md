# SentryBox Branding

This folder contains reusable SVG branding assets for Home Assistant and HACS:

- `icon.svg`: square icon for integration cards and listings
- `logo.svg`: wider wordmark/logo for docs, GitHub, or brand submissions

Notes:

- Home Assistant does not automatically load branding from this repository.
- For first-class branding in Home Assistant and HACS, submit the icon/logo to the Home Assistant brands repository.
- These files are vector SVGs, so they can be exported to PNG if a store or listing requires raster assets.
